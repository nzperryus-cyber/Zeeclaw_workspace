## SECTION 020-PNT — Paint Shop Record Book & Launch Requirements

### Key Phases
1. **Installation** — Contractor's period for installation
2. **Start-Up, Debug and Prove Out** — Contractor's checkout, adjustment; must complete 20-hour internal dry run before 4-hour wet buy-off
3. **Facility Launch** — Contractor proves system performs to design parameters (4-hour or equivalent wet run)
4. **Production Launch** — Belongs to the plant — includes training, system fill, job 1

### Contractor Responsibilities
- Appoint a **dedicated project engineer** for each system, responsible through all phases
- Develop a **launch plan** with Ford VO Project Engineer and plant
- Follow all applicable **Ford VO Specifications and Manufacturing Standards**
- Ensure personnel and all Sub-Contractors are aware of buy-off requirements at contract inception

### Project Record Book (3 books required)
1. **Cell Specification Book**
2. **Project Record File Book**
3. **Equipment Baseline Data Book**

Three (3) hard copies + 3 CDs — distributed to: Plant, Ford VO Project Co-Coordinator, FSC/PM/PE

### Record Book Structure
- **Section 1** (Ford responsibility): Project Initiation, Scope of Work, Bid Packages, Bid Award
- **Section 2** (Contractor responsibility): Design Engineering, Fabrication, Installation, Start-Up, Facility Launch, Production Launch

### Launch Sign-Off Requirements
- **Three sign-offs required** for each Baseline Data Section: Contractor, Specific Plant, Ford VO
- Baseline Data accepted on a **system basis** (not component-by-component)

### Start-Up Phase
- Contractor provides **full time electrical engineer + mechanical engineer per shift**
- **20-hour or equivalent performance run** required before official 4-hour wet buy-off
- Contractor must be **100% complete** with installation/tryout/debug before Facility Launch

### Training Requirements
- **Operator Level Training**
- **Electrical Maintenance Level Training**
- **Mechanical Maintenance Level Training**
- **Advanced Mechanical and Electrical Level Training**
- **Advanced System Level Training for Engineers and Skilled Trades**
- Instructor/student ratio requirements apply
- Hands-on training required
- Follow-up training required
